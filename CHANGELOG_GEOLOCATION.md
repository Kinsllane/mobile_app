## 🆕 Новая фича: "Книжные места" 🗺️

### Описание
Интерактивная карта для отметки мест, где пользователь читает книги. Использует **реальную геолокацию** устройства и **Google Maps**.

```bash
npm install expo-location react-native-maps
```

### Версии:
- `expo-location` - для получения GPS координат
- `react-native-maps` - для отображения карты

### 1. **ReadingPlacesScreen.js**
`src/screens/ReadingPlacesScreen.js`

**Что делает:**
- Отображает интерактивную карту

## ⚠️ Важные моменты

### Для разработки:
- ❗ Геолокация НЕ работает в веб-версии (только iOS/Android)
- ❗ Нужно тестировать на реальном устройстве или эмуляторе
- ❗ В первый раз система запросит разрешения

### Для Google Maps:
- На Android нужен Google Maps API ключ
- В `app.json` есть поле для API ключа
- Для теста можно использовать без ключа (с ограничениями)

### Разрешения:
- iOS: Разрешение запрашивается автоматически при первом использовании
- Android: Разрешение запрашивается автоматически

## 🎓 Объяснение для препода

### Почему эта фича?
**Вопрос:** Почему геолокация в приложении для книг?

**Ответ:** Люди читают книги в разных местах - дома, в кафе, в парке, в транспорте. Эта фича позволяет:
1. **Запоминать моменты** - связать книгу с местом и эмоциями
2. **Визуализировать опыт** - увидеть свою "читательскую карту"
3. **Делиться историями** - "Я прочитал эту книгу в удивительном кафе"

### Технологии:
- **Геолокация**: `expo-location` - получение GPS координат
- **Карты**: `react-native-maps` - интеграция Google Maps
- **Разрешения**: Корректный запрос системных разрешений
- **Хранение**: AsyncStorage для персистентности данных
- **UI/UX**: Модальные окна, интерактивные элементы

1. Показать запрос разрешений ✅

4. Показать маркер на карте ✅
