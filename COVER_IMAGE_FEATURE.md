# 📸 Обложки книг — Новая функция

## ✅ Реализовано

Добавлена полноценная функция загрузки и обрезки фото обложек книг!

---

## 🎯 Возможности

### 1. **Загрузка обложки**
- ✅ Выбор изображения из галереи
- ✅ Съёмка фото на камеру
- ✅ Превью текущей обложки

### 2. **Обрезка изображения**
Пользователь может выбрать один из 3 форматов:
- **Книга (2:3)** — классический формат обложки книги (300x450px)
- **Квадрат (1:1)** — квадратная обложка (300x300px)
- **Широкий (16:9)** — широкоформатная обложка (400x225px)

### 3. **Удаление обложки**
- Можно вернуться к цветной плашке с эмодзи

### 4. **Отображение**
- В списке книг показывается реальная обложка
- На экране деталей — полноразмерная обложка
- Если обложки нет — цветная плашка с эмодзи 📚

---

## 📦 Установленные пакеты

```bash
expo-image-picker      # Выбор изображений и съёмка
expo-image-manipulator # Обрезка и обработка изображений
```

## 📱 Компоненты

### `CoverImagePicker.js`
Новый компонент для управления обложкой:

**Возможности:**
- Превью текущей обложки
- 2 кнопки: "Выбрать из галереи" и "Сделать фото"
- Кнопка "Удалить обложку" (если есть)
- Модальное окно выбора формата обрезки
- Визуальная индикация обработки

**Props:**
- `currentImage` — URI текущего изображения
- `onImageSelected` — callback при выборе нового изображения
- `coverColor` — цвет плашки (если нет изображения)

## 🚀 Как использовать

### Добавление книги с обложкой:
1. Открыть "Добавить книгу"
2. Нажать "Выбрать из галереи" или "Сделать фото"
3. Выбрать изображение
4. Выбрать формат обрезки (Книга / Квадрат / Широкий)
5. Заполнить остальные поля
6. Нажать "Сохранить книгу"

### Редактирование обложки:
1. Открыть детали книги
2. Нажать "Редактировать книгу"
3. Изменить обложку через компонент
4. Сохранить изменения

### Удаление обложки:
1. В редактировании книги
2. Нажать "Удалить обложку"
3. Подтвердить удаление
4. Вернётся цветная плашка

## 📊 Технические детали

### Сохранение изображений
Изображения сохраняются как **URI** в структуре книги:
```js
{
  id: '123',
  title: 'Книга',
  author: 'Автор',
  coverImage: 'file:///path/to/image.jpg', // URI
  coverColor: '#FF6B6B', // Fallback цвет
  ...
}
```

### Обработка изображений
Используется `expo-image-manipulator` для:
- Изменения размера (resize)
- Сжатия (compress: 0.8)
- Конвертации в JPEG

### Разрешения
Разрешения запрашиваются автоматически при первом использовании:
- Камера — при нажатии "Сделать фото"
- Галерея — при нажатии "Выбрать из галереи"
