# ✨ Краткая сводка новых возможностей

## 🎯 Что добавлено (пункт препода: "несколько активити взаимодействующих друг с другом")

### 📱 Новые экраны (3 штуки):

#### 1. **📊 Экран статистики** (`StatisticsScreen`)
**Путь:** Главный экран → кнопка "📊 Статистика"

**Что показывает:**
- 4 красивые карточки с основными метриками
- Детальные прогресс-бары по статусам книг
- Топ-5 жанров с анимированными прогресс-барами
- Кнопка быстрого перехода к целям

**Взаимодействие:** 
- Получает данные из общего контекста (BooksContext)
- Переход к экрану целей одним нажатием

---

#### 2. **🎯 Экран целей чтения** (`GoalsScreen`)
**Путь:** Главный экран → кнопка "🎯 Цели"

**Что делает:**
- Установка годовой цели чтения
- Установка месячной цели чтения
- Отслеживание прогресса в реальном времени
- Показывает сколько книг осталось прочитать
- Поздравления при достижении целей! 🎉

**Взаимодействие:**
- Сохраняет цели в AsyncStorage
- Анализирует книги из BooksContext
- Переход к статистике и обратно к списку книг

---

#### 3. **❤️ Экран избранного** (`FavoritesScreen`)
**Путь:** Главный экран → кнопка "❤️ Избранное"

**Что делает:**
- Показывает только избранные книги
- Красивое пустое состояние если избранных нет
- Клик на книгу → переход к деталям

**Взаимодействие:**
- Фильтрует книги с флагом `isFavorite: true`
- Переход к деталям книги для просмотра/редактирования

---

## 🔗 Как экраны взаимодействуют:

### Пример 1: Полный цикл работы с книгой
```
1. BooksListScreen: выбираю книгу
   ↓
2. BookDetailScreen: добавляю в избранное ❤️
   ↓
3. Возврат на главный экран
   ↓
4. Клик "❤️ Избранное"
   ↓
5. FavoritesScreen: вижу избранную книгу!
```

### Пример 2: Работа с целями и статистикой
```
1. BooksListScreen: клик "🎯 Цели"
   ↓
2. GoalsScreen: ставлю цель 12 книг в год
   ↓
3. GoalsScreen: вижу прогресс 5/12 (42%)
   ↓
4. Клик "📊 Статистика"
   ↓
5. StatisticsScreen: детальная визуализация
   ↓
6. Клик "🎯 Перейти к целям"
   ↓
7. GoalsScreen: проверяю прогресс
   ↓
8. Клик "📚 К списку книг"
   ↓
9. BooksListScreen: возврат на главную
```

### Пример 3: Навигация туда-обратно
```
Главный ⟷ Статистика ⟷ Цели ⟷ Главный
   ↕
Детали книги
   ↕
Избранное
```

## 💾 Общее хранилище данных (Context API):

Все экраны используют **один источник истины** - `BooksContext`:
- `books[]` - список всех книг
- `addBook()` - добавить книгу
- `updateBook()` - обновить книгу (например, добавить в избранное)
- `deleteBook()` - удалить книгу
- `getBookById()` - получить книгу по ID

Изменение данных на одном экране **автоматически видно на всех остальных**!

Приложение демонстрирует:
