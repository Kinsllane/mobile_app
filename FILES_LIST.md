# 📁 Список файлов проекта

## 📱 Исходный код приложения

### Главный файл
- `App.js` - Главный компонент с настройкой навигации

### Экраны (src/screens/)
1. `BooksListScreen.js` - Главный экран со списком книг
2. `BookDetailScreen.js` - Детали книги с заметками
3. `AddBookScreen.js` - Добавление новой книги
4. `EditBookScreen.js` - Редактирование книги
5. `StatisticsScreen.js` - Детальная статистика
6. `GoalsScreen.js` - Управление целями чтения
7. `FavoritesScreen.js` - Избранные книги

### Компоненты (src/components/)
1. `StatisticsCard.js` - Карточка статистики для главного экрана
2. `ProgressBar.js` - Анимированный прогресс-бар
3. `StatCard.js` - Карточка метрики
4. `PieChart.js` - Круговая диаграмма (не используется в финале)
5. `BarChart.js` - Столбчатая диаграмма (не используется в финале)

### Context API (src/context/)
- `BooksContext.js` - Управление состоянием приложения
