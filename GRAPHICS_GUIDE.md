# 📊 Графика и визуализация данных

## 🎨 Добавленная графика

Приложение теперь включает **профессиональную визуализацию данных** с двумя типами диаграмм:

---

### 1. 🥧 Круговая диаграмма (Pie Chart)

**Что показывает:**
- Распределение книг по статусам
- Процентное соотношение
- Количество книг в каждой категории

**Цветовая схема:**
- 🟢 Зеленый - Прочитано
- 🟠 Оранжевый - Читаю
- 🟣 Фиолетовый - Планирую прочитать
- 🔴 Красный - Отложено

**Особенности:**
- Показываются только непустые категории
- Проценты отображаются прямо на диаграмме
- Интерактивная легенда с точными числами
- Плавные закругленные края
- Размер сегментов пропорционален количеству книг

---

### 2. 📊 Столбчатая диаграмма (Bar Chart)

**Что показывает:**
- Топ-8 жанров по популярности
- Количество книг каждого жанра
- Сравнение предпочтений

**Особенности:**
- Автоматическая сортировка (самые популярные сверху)
- Горизонтальная прокрутка для всех жанров
- Значения над каждым столбиком
- Цветовая дифференциация жанров
- Динамическая высота столбиков

---

## 🎯 Как использовать

### Шаг 1: Открыть статистику
1. На главном экране нажмите **"📊 Показать статистику"**
2. Появится блок со статистикой

### Шаг 2: Показать диаграммы
1. В блоке статистики нажмите **"📊 Показать диаграммы"**
2. Появятся две диаграммы:
   - Круговая диаграмма статусов
   - Столбчатая диаграмма жанров

### Шаг 3: Взаимодействие
- **Круговая диаграмма**: 
  - Проценты видны прямо на сегментах
  - Легенда показывает точные числа
  
- **Столбчатая диаграмма**:
  - Свайп влево/вправо для просмотра всех жанров
  - Числа над столбиками показывают количество

### Шаг 4: Скрыть
- Нажмите **"📊 Скрыть диаграммы"** чтобы освободить место

## 🛠 Технические детали

### Библиотеки:
- **react-native-svg**: Встроена в Expo SDK 54
- **Никаких сторонних библиотек** для графиков
- **Самописные компоненты**: PieChart и BarChart

### Производительность:
- ✅ Оптимизированный рендеринг
- ✅ Динамический пересчет при изменении данных
- ✅ Плавная работа даже с большим количеством книг

### Адаптивность:
- 📱 Работает на любом размере экрана
- 🔄 Горизонтальная прокрутка для длинных списков
- 🎨 Автоматическое скрытие мелких сегментов (<5%)
