**Типография:**

- Нет экстра-жирных весов (800-900 удалены для Android)

## ✅ Что ДОБАВЛЕНО

### Светлые цвета
- `colors.background` = `#FFFFFF`
- `colors.surface` = `#FFFFFF`
- `colors.textPrimary` = `#1F2937`
- `colors.border` = `#E5E7EB`

### Кроссплатформенные тени
```js
shadows.sm = {
  shadowColor: '#000',
  shadowOffset: { width: 0, height: 1 },
  shadowOpacity: 0.05,
  shadowRadius: 2,
  elevation: 1,
}
```

### Импорты theme.js
Все экраны и компоненты теперь импортируют:
```js
import { colors, typography, spacing, borderRadius, shadows } from '../utils/theme';
```
