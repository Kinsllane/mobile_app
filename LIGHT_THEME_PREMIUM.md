## 📐 Типография

### Размеры:
- **xs**: 12px (метаданные)
- **sm**: 14px (вторичный текст)
- **base**: 16px (основной текст)
- **lg**: 18px (подзаголовки)
- **xl**: 20px (заголовки карточек)
- **2xl**: 24px (заголовки экранов)
- **3xl**: 30px (крупные заголовки)

### Веса (кроссплатформенные):
- **regular**: 400
- **medium**: 500
- **semibold**: 600
- **bold**: 700

**Нет 800-900!** Для Android это не нужно.

## 📦 Новые файлы

### `src/utils/theme.js`
Центральный файл с дизайн-системой:
- `colors` - вся палитра
- `typography` - размеры и веса шрифтов
- `spacing` - отступы
- `borderRadius` - скругления
- `shadows` - кроссплатформенные тени

### Использование:
```javascript
import { colors, typography, spacing, borderRadius, shadows } from '../utils/theme';

const styles = StyleSheet.create({
  container: {
    backgroundColor: colors.background,
    padding: spacing.base,
    borderRadius: borderRadius.lg,
    ...shadows.md,
  },
  title: {
    fontSize: typography.xl,
    fontWeight: typography.bold,
    color: colors.textPrimary,
  },
});
```

## 🔧 Что исправлено

### Проблема 1: Чёрные квадраты на Android
**Было**: `shadowOpacity` не работает на Android, показывались артефакты  
**Решение**: Используем `elevation` для Android, `shadow*` для iOS

### Проблема 2: Градиенты выглядят плохо
**Было**: `LinearGradient` создавал лишний визуальный шум  
**Решение**: Убрали градиенты, используем чистые цвета

### Проблема 3: Слишком яркие цвета
**Было**: Неоновые `#00F5FF`, `#FF006E` слишком кричат  
**Решение**: Профессиональная палитра с правильным контрастом

### Проблема 4: Разная верстка на платформах
**Было**: Разные шрифты, тени, отступы  
**Решение**: Единая дизайн-система через `theme.js`
