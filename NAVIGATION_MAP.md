## 📋 Подробное описание переходов

### 🏠 BooksListScreen (Главная)
**Исходящие переходы:**
1. **Кнопка "📊 Статистика"** → StatisticsScreen
2. **Кнопка "🎯 Цели"** → GoalsScreen  
3. **Кнопка "❤️ Избранное"** → FavoritesScreen
4. **Клик на карточку книги** → BookDetailScreen (передаёт `bookId`)
5. **Кнопка "+ Добавить книгу"** → AddBookScreen

**Входящие переходы:**
- Назад из всех экранов
- Кнопка "📚 К списку книг" из GoalsScreen

---

### 📊 StatisticsScreen (Статистика)
**Исходящие переходы:**
1. **Кнопка "🎯 Перейти к целям чтения"** → GoalsScreen
2. **Кнопка назад** → BooksListScreen

**Входящие переходы:**
- Кнопка "📊 Статистика" из BooksListScreen
- Кнопка "📊 Статистика" из GoalsScreen

**Данные:**
- Читает: `books` из BooksContext
- Вычисляет: статистику, топ-жанры, распределение

---

### 🎯 GoalsScreen (Цели чтения)
**Исходящие переходы:**
1. **Кнопка "📊 Статистика"** → StatisticsScreen
2. **Кнопка "📚 К списку книг"** → BooksListScreen
3. **Кнопка назад** → предыдущий экран

**Входящие переходы:**
- Кнопка "🎯 Цели" из BooksListScreen
- Кнопка "🎯 Перейти к целям" из StatisticsScreen

**Данные:**
- Читает: `books` из BooksContext
- Сохраняет: цели в AsyncStorage (`@reading_goals`)
- Вычисляет: прогресс по годам и месяцам

---

### ❤️ FavoritesScreen (Избранное)
**Исходящие переходы:**
1. **Клик на карточку книги** → BookDetailScreen (передаёт `bookId`)
2. **Кнопка "📚 К списку книг"** → BooksListScreen (если пусто)
3. **Кнопка назад** → BooksListScreen

**Входящие переходы:**
- Кнопка "❤️ Избранное" из BooksListScreen

**Данные:**
- Читает: `books` из BooksContext
- Фильтрует: только книги с `isFavorite: true`

---

### 📖 BookDetailScreen (Детали книги)
**Исходящие переходы:**
1. **Кнопка "Редактировать"** → EditBookScreen (передаёт `bookId`)
2. **Кнопка "Удалить книгу"** → удаление + возврат на BooksListScreen
3. **Кнопка назад** → предыдущий экран

**Входящие переходы:**
- Клик на книгу из BooksListScreen
- Клик на книгу из FavoritesScreen
- Возврат из EditBookScreen

**Данные:**
- Получает: `bookId` из route.params
- Читает: `getBookById(bookId)` из BooksContext
- Изменяет: `isFavorite`, `notes`, `keyPoints` через `updateBook()`

**Модальные окна:**
- Редактирование заметок
- Редактирование ключевых моментов

---

### ✏️ EditBookScreen (Редактирование)
**Исходящие переходы:**
1. **Кнопка "Сохранить"** → обновление + возврат на BookDetailScreen
2. **Кнопка "Отмена"** → возврат без сохранения

**Входящие переходы:**
- Кнопка "Редактировать" из BookDetailScreen

**Данные:**
- Получает: `bookId` из route.params
- Читает: текущие данные книги
- Изменяет: все поля книги через `updateBook()`

---

### ➕ AddBookScreen (Добавление)
**Исходящие переходы:**
1. **Кнопка "Сохранить"** → создание книги + возврат на BooksListScreen
2. **Кнопка "Отмена"** → возврат без сохранения

**Входящие переходы:**
- Кнопка "+ Добавить книгу" из BooksListScreen

**Данные:**
- Создаёт: новую книгу через `addBook()`
- Генерирует: `id`, `coverColor`, устанавливает значения по умолчанию

---

## 🔄 Циклы взаимодействия

### Цикл 1: Статистика ↔ Цели
```
BooksListScreen 
    → "📊 Статистика" → StatisticsScreen 
    → "🎯 Перейти к целям" → GoalsScreen 
    → "📊 Статистика" → StatisticsScreen 
    → "Назад" → BooksListScreen
```

### Цикл 2: Редактирование книги
```
BooksListScreen 
    → клик на книгу → BookDetailScreen 
    → "Редактировать" → EditBookScreen 
    → "Сохранить" → BookDetailScreen 
    → "Назад" → BooksListScreen
```

### Цикл 3: Избранное
```
BooksListScreen 
    → клик на книгу → BookDetailScreen 
    → нажать ❤️ (добавить в избранное) 
    → "Назад" → BooksListScreen 
    → "❤️ Избранное" → FavoritesScreen 
    → видим добавленную книгу
    → клик на книгу → BookDetailScreen
    → нажать ❤️ (убрать из избранного)
    → "Назад" → FavoritesScreen
    → книга исчезла
```

## 🗂️ Передаваемые данные

### Navigation params (параметры навигации)

**BookDetailScreen:**
```javascript
navigation.navigate('BookDetail', { bookId: '123' })
// Получение: const { bookId } = route.params
```

**EditBookScreen:**
```javascript
navigation.navigate('EditBook', { bookId: '123' })
// Получение: const { bookId } = route.params
```

### Context API (общее состояние)

**Все экраны имеют доступ к:**
```javascript
{
  books: Array,           // Массив всех книг
  loading: Boolean,       // Состояние загрузки
  addBook: Function,      // Добавить книгу
  updateBook: Function,   // Обновить книгу
  deleteBook: Function,   // Удалить книгу
  getBookById: Function   // Получить книгу по ID
}
```

### AsyncStorage (постоянное хранилище)

**Ключи:**
- `@books_storage` - все книги (используют все экраны через Context)
- `@reading_goals` - цели чтения (только GoalsScreen)

          │                  │                  │
          └──────────────────┼──────────────────┘

**Изменение данных:**
1. Пользователь изменяет книгу на EditBookScreen
2. EditBookScreen вызывает `updateBook(bookId, newData)`
3. BooksContext обновляет массив `books`
4. BooksContext сохраняет в AsyncStorage
5. Все экраны автоматически видят обновлённые данные

## ✨ Особенности навигации

### 1. Стек навигации (Native Stack)
- Каждый новый экран добавляется в стек
- Кнопка "Назад" убирает верхний экран из стека
- История переходов сохраняется

### 2. Передача данных
- `navigation.navigate(screenName, params)` - переход с данными
- `route.params` - получение переданных данных

### 3. Возврат назад
- `navigation.goBack()` - программный возврат
- Аппаратная кнопка "Назад" на Android
- Жест свайпа на iOS

### 4. Заголовки экранов
```javascript
'BooksList' → 'Мои Книги'
'BookDetail' → 'Детали книги'
'AddBook' → 'Добавить книгу'
'EditBook' → 'Редактировать книгу'
'Statistics' → 'Статистика'
'Goals' → 'Цели чтения'
'Favorites' → 'Избранное'
```
