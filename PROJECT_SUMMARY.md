## 📱 Экраны (Activity) ⭐ Требование преподавателя

### Всего экранов: 7

1. **BooksListScreen** (Главный)
   - Список всех книг с фильтрацией
   - Поиск и сортировка
   - Быстрая навигация к другим экранам
   - Добавление в избранное прямо с карточек

2. **BookDetailScreen** (Детали книги)
   - Полная информация о книге
   - Редактирование заметок
   - Редактирование ключевых моментов
   - Переключение избранного
   - Переход к редактированию
   - Удаление книги

3. **AddBookScreen** (Добавление)
   - Форма создания новой книги
   - Выбор жанра

   - Рейтинг 1-5 звёзд

4. **EditBookScreen** (Редактирование)
   - Изменение всех полей книги
   - Сохранение изменений
   - Отмена без сохранения

5. **StatisticsScreen** (Статистика) ⭐ НОВЫЙ
   - Детальная аналитика
   - 6 карточек метрик
   - Прогресс-бары по статусам
   - Топ-5 жанров
   - Переход к целям

6. **GoalsScreen** (Цели чтения) ⭐ НОВЫЙ
   - Установка годовой цели
   - Установка месячной цели
   - Отслеживание прогресса
   - Визуализация достижений
   - Навигация к статистике

7. **FavoritesScreen** (Избранное) ⭐ НОВЫЙ
   - Список избранных книг
   - Фильтрация по isFavorite
   - Переход к деталям
   - Пустое состояние с подсказкой

## 🔄 Взаимодействие экранов

### Типы взаимодействия:

1. **Навигация (Navigation)**
   - Прямые переходы между экранами
   - Передача данных через params
   - Обратная навигация (кнопка "Назад")

2. **Общее состояние (Context API)**
   - BooksContext доступен на всех экранах
   - Изменения синхронизируются автоматически
   - Реактивные обновления UI

3. **Постоянное хранилище (AsyncStorage)**
   - Книги: `@books_storage`
   - Цели: `@reading_goals`
   - Автосохранение при изменениях

### Примеры взаимодействия:

**Цикл 1: Добавление в избранное**
```
BooksListScreen 
  → клик на ❤️ на карточке 
  → updateBook через Context 
  → AsyncStorage обновляется 
  → FavoritesScreen автоматически видит изменения
```

**Цикл 2: Статистика ↔ Цели**
```
BooksListScreen 
  → кнопка "Статистика" 
  → StatisticsScreen 
  → кнопка "Перейти к целям" 
  → GoalsScreen 
  → кнопка "Статистика" 
  → StatisticsScreen 
  → кнопка "Назад" 
  → BooksListScreen
```

**Цикл 3: Редактирование книги**
```
BooksListScreen 
  → клик на книгу (передаётся bookId) 
  → BookDetailScreen 
  → кнопка "Редактировать" (передаётся bookId) 
  → EditBookScreen 
  → изменение данных 
  → updateBook через Context 
  → возврат на BookDetailScreen 
  → данные обновлены
  → возврат на BooksListScreen
  → изменения видны в списке
```
