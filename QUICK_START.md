# 🚀 Быстрый старт - Book Hobby App

## Для преподавателя: Как запустить приложение

### ✅ Вариант 1: Expo Go (самый простой)

1. **Установите Expo Go на телефон**
   - Android: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - iOS: [App Store](https://apps.apple.com/app/expo-go/id982107779)

2. **Запустите проект на компьютере:**
   ```bash
   cd book-hobby-app
   npm start
   ```

3. **Отсканируйте QR-код**
   - Android: в приложении Expo Go
   - iOS: в стандартной камере телефона

4. **Готово!** Приложение загрузится на телефоне

---

### ✅ Вариант 2: Эмулятор Android (если есть Android Studio)

1. **Запустите эмулятор Android** из Android Studio

2. **В терминале выполните:**
   ```bash
   cd book-hobby-app
   npm run android
   ```

3. **Готово!** Приложение установится на эмуляторе

## 🔄 Проверка взаимодействия экранов

### Сценарий 1: Добавление в избранное и проверка
```
1. Главный экран → выберите книгу "Атомные привычки"
2. Экран деталей → нажмите "🤍 Добавить в избранное"
3. Вернитесь на главный экран (стрелка назад)
4. Нажмите кнопку "❤️ Избранное"
5. ✅ Обе книги (Sapiens и Атомные привычки) в избранном!
```

### Сценарий 2: Постановка цели и отслеживание
```
1. Главный экран → нажмите "🎯 Цели"
2. Экран целей → нажмите "✏️ Изменить" для годовой цели
3. Введите "10" → нажмите "Сохранить"
4. ✅ Видите прогресс: 2/10 книг (20%)
5. Нажмите "📊 Статистика"
6. ✅ Видите те же данные в детальной статистике
7. Нажмите стрелку "Назад"
8. Нажмите "📚 К списку книг"
9. ✅ Вернулись на главный экран
```

### Сценарий 3: Добавление книги и проверка статистики
```
1. Главный экран → нажмите "+ Добавить книгу"
2. Заполните форму:
   - Название: "1984"
   - Автор: "Джордж Оруэлл"
   - Жанр: "Классика"
   - Рейтинг: 5 звёзд
   - Статус: "Прочитано"
3. Нажмите "Сохранить"
4. ✅ Книга в списке!
5. Нажмите "📊 Статистика"
6. ✅ Счётчики обновились! (Всего книг: 4, Прочитано: 3)
7. ✅ В топ-жанрах появилась "Классика"
```

### Сценарий 4: Полный цикл редактирования
```
1. Главный экран → выберите книгу "Sapiens"
2. Экран деталей → нажмите "Редактировать" (рядом с Заметками)
3. Введите текст: "Потрясающая книга о человечестве!"
4. Нажмите "Сохранить"
5. ✅ Заметка сохранена и видна на экране деталей
6. Нажмите "Редактировать" книгу (верхняя кнопка)
7. Измените рейтинг на 4 звезды
8. Нажмите "Сохранить"
9. ✅ Рейтинг изменился
10. Вернитесь на главный → нажмите "📊 Статистика"
11. ✅ Средний рейтинг пересчитался!
```

## 🐛 Если что-то не работает

### Приложение не загружается
```bash
# Очистите кэш
npx expo start -c
```

### Ошибка с зависимостями
```bash
# Переустановите зависимости
rm -rf node_modules
npm install
```

### Телефон не видит QR-код
- Убедитесь, что телефон и компьютер в одной сети Wi-Fi
- Попробуйте переключиться с Tunnel на LAN (в терминале нажмите `s`)
