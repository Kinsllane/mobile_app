# 🚀 Быстрый тест нового дизайна

## Запуск приложения

```bash
# В папке проекта выполните:
npm start

# Или если npm start не работает:
npx expo start
```

## Что проверить за 2 минуты

### 1️⃣ Главный экран (10 сек)
- ✨ Карточки книг округлённые (20px)
- ❤️ Кнопка избранного в углу каждой карточки (большая, белая с тенью)
- 🎨 Три кнопки сверху: фиолетовая, оранжевая, красная

### 2️⃣ Детали книги (10 сек)
- Нажмите на любую книгу
- ❤️ Большая кнопка избранного под названием
- ✏️ Нажмите "Редактировать" у заметок - модальное окно с новым дизайном

### 3️⃣ Статистика (10 сек)
- Фиолетовый хедер
- 4 карточки метрик с цветными границами слева
- Прогресс бары с новыми цветами

### 4️⃣ Цели (10 сек)
- Оранжевый хедер
- Карточки целей
- Кнопки "Изменить" открывают модалки

### 5️⃣ Избранное (10 сек)
- Красный хедер
- Список избранных книг
- Сердечко ❤️ в каждой карточке

## Ключевые визуальные изменения

## Если что-то не так

1. Убедитесь что установлены все зависимости:
```bash
npm install
```

2. Очистите кеш:
```bash
npx expo start --clear
```

3. Проверьте что используете Expo SDK 54
