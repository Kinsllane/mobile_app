# 🗺️ КНИЖНЫЕ МЕСТА - Гайд по функции

## 📍 Что это?

**"Книжные места"** - это новая фича, которая позволяет отмечать на карте места, где вы читали книги. Используется **геолокация** и **интерактивная карта**.

---

## ✨ Возможности

### 1. Интерактивная карта
- 🗺️ Показывает все места, где вы читали книги
- 📍 Маркеры с цветами обложек книг
- 🧭 Отображение вашего текущего местоположения
- 🎯 Приближение и управление картой

### 2. Добавление мест
- ✅ Выбор книги из вашей библиотеки
- 📍 Автоматическое сохранение текущей геолокации
- 📝 Добавление заметки о месте (опционально)
- 🕒 Автоматическое сохранение даты и времени

### 3. Информация о местах
- 📖 Название книги и автор
- 📝 Ваша заметка о месте
- 📅 Дата добавления места
- 🎨 Цветной маркер (цвет обложки книги)

### 4. Управление местами
- 👆 Нажатие на маркер показывает детали
- 🗑️ Удаление места с подтверждением
- 📊 Счетчик добавленных мест

## 🚀 Как использовать

### Шаг 1: Открыть карту
1. На главном экране нажмите кнопку **"🗺️ Книжные места на карте"**
2. Приложение запросит доступ к геолокации (разрешите!)
3. Карта загрузится с вашим текущим местоположением

### Шаг 2: Добавить место
1. Нажмите кнопку **"📍 Добавить место"**
2. Выберите книгу из списка
3. Добавьте заметку (например: "Читал в кафе у дома")
4. Нажмите **"Сохранить"**
5. Место появится на карте!

### Шаг 3: Просмотр мест
1. Нажмите на любой маркер на карте
2. Увидите всплывающую подсказку с названием книги
3. Нажмите на подсказку для полной информации
4. Можете удалить место через всплывающее окно

## 📦 Технические детали

### Установленные пакеты
```bash
npm install expo-location react-native-maps
```

### Используемые технологии
- **expo-location**: Получение геолокации пользователя
- **react-native-maps**: Интерактивная карта с маркерами
- **AsyncStorage**: Сохранение мест локально
- **LinearGradient**: Красивые градиенты на кнопках

### Разрешения в app.json
- `ACCESS_COARSE_LOCATION` (Android)
- `ACCESS_FINE_LOCATION` (Android)
- `NSLocationWhenInUseUsageDescription` (iOS)

### Хранение данных
Места сохраняются локально в AsyncStorage:
```javascript
{
  id: "timestamp",
  bookId: "book_id",
  bookTitle: "Название книги",
  bookAuthor: "Автор",
  bookColor: "#FF6B6B",
  latitude: 55.751244,
  longitude: 37.618423,
  note: "Заметка о месте",
  timestamp: "2026-06-23T..."
}
```

## 🔧 Troubleshooting

### Проблема: Карта не загружается
**Решение**: 
1. Проверьте разрешение на геолокацию
2. Включите GPS на устройстве
3. Перезапустите приложение

### Проблема: "Нет доступа к геолокации"
**Решение**:
1. Откройте настройки телефона
2. Найдите приложение
3. Разрешите доступ к местоположению

### Проблема: Маркеры не отображаются
**Решение**:
1. Убедитесь, что добавили места
2. Проверьте счетчик мест вверху экрана
3. Попробуйте приблизить карту

### Почему эта фича подходит для книжного приложения?
