# 📚 Хобби-приложение для обзора книг

Кроссплатформенное мобильное приложение на React Native с Expo SDK 54 для учета и обзора прочитанных книг.

## 🚀 Возможности

### ✅ Основные функции
- **CRUD операции**: Добавление, просмотр, редактирование и удаление книг
- **Постоянное хранилище**: Все данные сохраняются на устройстве с помощью AsyncStorage
- **Детальная информация**: Название, автор, рейтинг, статус чтения, краткая сводка
- **16 жанров**: От фантастики до бизнес-литературы

### 🔍 Поиск и фильтрация
- **Быстрый поиск** по названию и автору книги
- **Фильтрация по статусу**: Все / Планирую прочитать / Читаю / Прочитано / Отложено
- **Фильтрация по жанрам**: 16 доступных жанров
- **Счетчик результатов**: Показывает, сколько книг найдено
- **Сворачиваемые фильтры** для экономии места

### 🔄 Сортировка
- **По дате добавления** (новые сверху)
- **По рейтингу** (лучшие сверху)
- **По названию** (А-Я)
- **По автору** (А-Я)

### 📊 Статистика чтения
- **Основные метрики**: Карточки с ключевыми показателями
- **Анимированные прогресс-бары** для визуализации
- **Распределение по статусам**: Прочитано / Читаю / Планирую / Отложено
- **Топ-5 жанров** с цветовой кодировкой
- **Детальный экран статистики** с полной аналитикой

### 🎯 Цели чтения
- **Годовая цель**: Установите сколько книг хотите прочитать в год
- **Месячная цель**: Краткосрочная мотивация
- **Отслеживание прогресса** в реальном времени
- **Поздравления** при достижении целей 🎉

### ❤️ Избранное
- **Добавление в избранное** с экрана деталей книги
- **Отдельный экран** для быстрого доступа к любимым книгам
- **Визуальное отличие** с сердечками ❤️

### 📝 Персональные заметки
- **Заметки по книге**: Добавляйте личные мысли и впечатления
- **Ключевые моменты**: Список важных идей из книги (каждая с новой строки)
- **Модальные окна** для удобного редактирования
- **Автосохранение** в AsyncStorage

### 📱 7 взаимодействующих экранов (Activity)
1. **BooksListScreen** - Главный экран со списком книг
2. **BookDetailScreen** - Детали книги с заметками
3. **AddBookScreen** - Добавление новой книги
4. **EditBookScreen** - Редактирование книги
5. **StatisticsScreen** - Детальная статистика и аналитика
6. **GoalsScreen** - Управление целями чтения
7. **FavoritesScreen** - Избранные книги

**Все экраны взаимодействуют через:**
- Context API для общего состояния
- React Navigation для передачи данных
- AsyncStorage для постоянного хранилища

📋 **Подробная схема взаимодействия экранов в файле [`ACTIVITIES_GUIDE.md`](./ACTIVITIES_GUIDE.md)**

## 📱 Запуск приложения

### Вариант 1: С Expo Go (рекомендуется для SDK 54)

1. Установите Expo Go на ваш телефон из магазина приложений
2. Убедитесь, что версия Expo Go поддерживает SDK 54

3. Запустите проект:
```bash
cd book-hobby-app
npm start
```

4. Отсканируйте QR-код в приложении Expo Go

### Вариант 2: Отладка по USB с Android SDK

Если версия Expo Go не совпадает, можно настроить отладку через USB:

#### Установка Android SDK

1. **Установите Android Studio:**
   - Скачайте с https://developer.android.com/studio
   - Запустите установщик и следуйте инструкциям

2. **Настройте Android SDK:**
   - Откройте Android Studio
   - Перейдите в Tools → SDK Manager
   - Установите необходимые компоненты:
     - Android SDK Platform (последняя версия)
     - Android SDK Build-Tools
     - Android SDK Command-line Tools
     - Android SDK Platform-Tools

3. **Добавьте переменные окружения:**
   
   В PowerShell (от администратора):
   ```powershell
   [System.Environment]::SetEnvironmentVariable("ANDROID_HOME", "C:\Users\ВашеИмя\AppData\Local\Android\Sdk", "User")
   ```
   
   Добавьте в PATH:
   ```powershell
   $path = [System.Environment]::GetEnvironmentVariable("Path", "User")
   $androidPath = ";C:\Users\ВашеИмя\AppData\Local\Android\Sdk\platform-tools;C:\Users\ВашеИмя\AppData\Local\Android\Sdk\tools"
   [System.Environment]::SetEnvironmentVariable("Path", $path + $androidPath, "User")
   ```

4. **Включите отладку по USB на телефоне:**
   - Перейдите в Настройки → О телефоне
   - Нажмите 7 раз на "Номер сборки"
   - Вернитесь и откройте "Для разработчиков"
   - Включите "Отладка по USB"

5. **Подключите телефон и проверьте:**
   ```bash
   adb devices
   ```
   
   Вы должны увидеть ваше устройство в списке.

6. **Запустите приложение:**
   ```bash
   npm run android
   ```

## 🐛 Возможные проблемы

### Проблема: Версия Expo Go не совпадает
**Решение:** Используйте отладку по USB с Android SDK (см. выше)

### Проблема: `adb devices` не находит устройство
**Решение:** 
- Проверьте драйвера USB
- Убедитесь, что отладка по USB включена
- Попробуйте другой USB кабель/порт

### Проблема: Ошибки peer dependencies
**Решение:** Используйте `npm install --legacy-peer-deps`
