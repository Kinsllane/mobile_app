# 🚀 Как запустить приложение

## Быстрый старт (1 минута)

### 1. Откройте терминал в папке проекта:
```bash
cd book-hobby-app
```

### 2. Запустите проект:
```bash
npm start
```

### 3. На телефоне:
- **Android:** Откройте Expo Go → отсканируйте QR-код
- **iOS:** Откройте Камеру → отсканируйте QR-код

### 4. Готово! ✅

---

## Если не работает

### Очистить кэш:
```bash
npx expo start -c
```

### Переустановить зависимости:
```bash
rm -rf node_modules
npm install
npm start
```

### Проблема с Expo Go версией:
```bash
npm run android
```
(требуется Android Studio или эмулятор)

---

## Полезные команды

```bash
# Остановить сервер
Ctrl + C

# Очистить кэш и запустить
npx expo start -c

# Запуск на Android (требуется эмулятор)
npm run android

# Запуск на iOS (только Mac)
npm run ios

# Просмотр зависимостей
npm list
```

---

## Требования

- ✅ Node.js 14+ установлен
- ✅ npm или yarn установлен
- ✅ Expo Go на телефоне (или эмулятор)
- ✅ Телефон и компьютер в одной сети Wi-Fi
