## ✨ Основные фичи

### 1. 📖 Управление библиотекой книг
- Добавление книг с информацией
- Редактирование данных книги
- Удаление книг
- Рейтинг от 1 до 5 звёзд
- Статусы: Читаю, Прочитано, Планирую, Отложено
- 16 жанров на выбор
- Избранное ❤️

### 2. 🔍 Поиск и фильтрация
- Поиск по названию и автору
- Фильтр по статусу
- Фильтр по жанру
- Сортировка: по дате, рейтингу, названию, автору
- Счётчик активных фильтров

### 3. 📝 Заметки и ключевые моменты
- Добавление заметок к книгам
- Ключевые моменты (bullet points)
- Краткая сводка книги
- Редактирование через модальные окна

### 4. 📊 Статистика
- Общее количество книг
- Прочитанные книги
- Средний рейтинг
- Прогресс чтения
- Распределение по статусам
- Топ-5 жанров
- Визуализация через прогресс-бары

### 5. 🎯 Цели чтения
- Годовая цель
- Месячная цель
- Отслеживание прогресса
- Счётчик оставшихся книг
- Уведомление о достижении цели

### 6. ❤️ Избранное
- Список избранных книг
- Быстрый доступ
- Визуальное выделение

### 7. 🗺️ **НОВОЕ!** Книжные места (Геолокация)
- Интерактивная карта
- Отметка мест чтения с GPS координатами
- Заметки о местах
- Цветные маркеры (цвет обложки)
- Просмотр и удаление мест
- Счётчик добавленных мест

## 🎯 Экраны приложения

### 1. Список книг (BooksList)
- Главный экран
- Кнопки быстрого доступа (3 кнопки)
- Кнопка "Книжные места" 🗺️
- Поиск
- Фильтры и сортировка
- Список книг с карточками
- Кнопка "Добавить книгу"

### 2. Детали книги (BookDetail)
- Обложка книги (emoji)
- Название, автор, жанр, рейтинг
- Кнопка избранного ❤️
- Краткая сводка
- Ключевые моменты (редактируемые)
- Заметки (редактируемые)
- Кнопка "Редактировать"
- Кнопка "Удалить"

### 3. Добавление книги (AddBook)
- Форма с полями:
  - Название *
  - Автор *
  - Рейтинг (1-5)
  - Статус (4 варианта)
  - Жанр (16 вариантов)
  - Краткая сводка
- Кнопка "Сохранить"
- Кнопка "Отмена"

### 4. Редактирование (EditBook)
- Аналогично AddBook
- Предзаполненные поля
- Кнопка "Сохранить изменения"

### 5. Статистика (Statistics)
- Градиентный заголовок
- 4 карточки метрик (2x2):
  - Всего книг
  - Прочитано
  - Средний рейтинг
  - Прогресс %
- 2 доп. карточки:
  - Высокий рейтинг (4+)
  - Читаю сейчас
- Прогресс-бары по статусам
- Топ-5 жанров
- Кнопка "Перейти к целям"

### 6. Цели (Goals)
- Градиентный заголовок
- Общая статистика (3 показателя)
- Годовая цель с прогрессом
- Месячная цель с прогрессом
- Кнопки редактирования целей
- Модальное окно для изменения
- 2 кнопки навигации

### 7. Избранное (Favorites)
- Градиентный заголовок
- Счётчик избранных книг
- Список избранных карточек
- Пустое состояние с кнопкой
- Цветные маркеры книг

### 8. 🗺️ **НОВЫЙ!** Книжные места (ReadingPlaces)
- Полноэкранная интерактивная карта
- Маркеры мест на карте
- Инфо панель со счётчиком
- Кнопка "Добавить место"
- Модальное окно:
  - Выбор книги (скроллируемый список)
  - Поле для заметки
  - Кнопки Отмена/Сохранить
- Всплывающие подсказки на маркерах
- Возможность удаления мест

## 💾 Хранение данных

### AsyncStorage ключи
- `@books_storage` - список книг
- `@reading_goals` - цели чтения
- `@reading_places` - места чтения (с GPS координатами)

### Формат книги
```javascript
{
  id: "timestamp",
  title: "Название",
  author: "Автор",
  rating: 5,
  status: "Прочитано",
  genre: "Саморазвитие",
  summary: "Краткая сводка...",
  coverColor: "#FF6B6B",
  isFavorite: true,
  notes: "Мои заметки...",
  keyPoints: ["Пункт 1", "Пункт 2"]
}
```

### Формат места
```javascript
{
  id: "timestamp",
  bookId: "book_id",
  bookTitle: "Название книги",
  bookAuthor: "Автор",
  bookColor: "#FF6B6B",
  latitude: 55.751244,
  longitude: 37.618423,
  note: "Читал в кафе",
  timestamp: "2026-06-23T12:00:00"
}
```
