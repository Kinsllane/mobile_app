# 📖 Руководство пользователя

## 🚀 Быстрый старт

### Главный экран

При открытии приложения вы видите:
- **Кнопку статистики** (📊) - нажмите, чтобы посмотреть вашу статистику чтения
- **Поиск** - для быстрого поиска книг
- **Сортировку** - выберите, как отображать книги
- **Фильтры** - покажите только нужные книги по статусу
- **Список книг** - основная область с вашими книгами
- **Кнопку "+"** - для добавления новой книги

---

## 📚 Работа с книгами

### Добавление книги
1. Нажмите **"+ Добавить книгу"** внизу экрана
2. Заполните:
   - **Название** (обязательно)
   - **Автор** (обязательно)
   - **Рейтинг** - от 1 до 5 (необязательно)
   - **Статус** - выберите один из 4 вариантов
   - **Краткая сводка** - о чем книга (необязательно)
3. Нажмите **"💾 Сохранить книгу"**

### Просмотр деталей
- Нажмите на любую книгу в списке
- Увидите: рейтинг, статус, сводку, ключевые моменты, заметки

### Редактирование книги
**Вариант 1: Основная информация**
1. Откройте книгу
2. Нажмите **"✏️ Редактировать книгу"**
3. Измените нужные поля
4. Сохраните

**Вариант 2: Заметки**
1. Откройте книгу
2. Найдите секцию "Мои заметки"
3. Нажмите **"✏️ Редактировать"**
4. Напишите свои мысли и впечатления
5. Нажмите "Сохранить"

**Вариант 3: Ключевые моменты**
1. Откройте книгу
2. Найдите секцию "Ключевые моменты"
3. Нажмите **"✏️ Редактировать"**
4. Напишите каждый момент **с новой строки**
5. Нажмите "Сохранить"

### Удаление книги
1. Откройте книгу
2. Пролистайте вниз
3. Нажмите **"🗑️ Удалить книгу"**
4. Подтвердите удаление

## 🔍 Поиск и фильтрация

### Поиск
- Введите название книги или автора в строку поиска
- Результаты обновятся автоматически
- Нажмите **✕** чтобы очистить поиск

### Фильтры по статусу
Выберите один из вариантов:
- **Все** - показать все книги
- **Планирую прочитать** - только запланированные
- **Читаю** - книги, которые читаете сейчас
- **Прочитано** - завершенные книги
- **Отложено** - отложенные книги

### Сортировка
Выберите способ сортировки:
- **📅 Дата** - новые книги сверху (по умолчанию)
- **⭐ Рейтинг** - лучшие книги сверху
- **🔤 Название** - алфавитный порядок (А-Я)
- **✍️ Автор** - по автору (А-Я)

## 📊 Статистика

### Как открыть
Нажмите кнопку **"📊 Показать статистику"** вверху главного экрана

### Что показывает
- **Всего книг** - размер вашей библиотеки
- **Прочитано** - сколько книг завершено
- **Средний рейтинг** - средняя оценка прочитанных книг
- **Прогресс-бар** - какой процент книг прочитан
- **Детализация** - сколько читаете, планируете, отложили

### Как скрыть
Нажмите **"📊 Скрыть статистику"**

## 💡 Советы и трюки

### 🎯 Организация коллекции
1. **Используйте рейтинги** - ставьте оценки, чтобы быстро находить лучшие книги
2. **Обновляйте статусы** - переводите книги из "Читаю" в "Прочитано"
3. **Пишите заметки сразу** - пока впечатления свежие

### 🔍 Быстрый поиск
- Введите **несколько букв** из названия - не нужно печатать полностью
- Ищите по **фамилии автора** - быстрее, чем пролистывать

### 📝 Ключевые моменты
- Пишите **короткие тезисы** - так их проще перечитывать
- Используйте **свои слова** - так лучше запоминается
- Добавляйте **практические советы** - что можно применить сразу

### 📊 Мотивация
- Открывайте **статистику раз в неделю** - видеть прогресс мотивирует
- Ставьте **цели** - например, "Прочитать 20 книг в этом году"
- Следите за **средним рейтингом** - выбирайте книги получше

## ❓ Частые вопросы

**Q: Где хранятся мои данные?**  
A: Локально на вашем устройстве. Данные не передаются в интернет.

**Q: Могу ли я потерять данные?**  
A: Нет, они сохраняются автоматически после каждого изменения.

**Q: Как сделать резервную копию?**  
A: В текущей версии нет функции экспорта. Данные привязаны к устройству.

**Q: Сколько книг можно добавить?**  
A: Теоретически неограниченно. Приложение оптимизировано для 500+ книг.

**Q: Можно ли добавить обложки книг?**  
A: В текущей версии - нет. Запланировано в будущих обновлениях.

**Q: Есть ли синхронизация между устройствами?**  
A: Пока нет. Каждое устройство хранит свою коллекцию.
